Testes de API - Eco Farma

Essa página tem o intuito de testar todas as funções que serão usadas no site Eco Farma. Cada função cada funcionalidade será explicada para quê serve e também como é testada. Para fazer o teste, é necessário a API estar ligada, selecione o modo 'http' para rodar normalmente. Para testar a API, além de rodar em http, use o banco PostgreSql para fazer as consultas.
